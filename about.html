サービス概要

「Netflix 評価ランキング」は、UNIIS公式通販が企画・開発・運営するLINEミニアプリです。Netflix Japanで配信中の約8,000作品をIMDb（Internet Movie Database）の評価スコア順にランキング表示し、ユーザーが次に観る作品を簡単に見つけられるサービスを提供しています。

主な機能

  • IMDb評価ランキング — 全作品をIMDbスコア順に表示
  • ジャンル別ブラウズ — ドラマ、コメディ、アクション等27カテゴリ
  • 多条件フィルタリング — 国（米国・日本・韓国・欧州等）、年代、評価範囲、評価数で絞り込み
  • ソート機能 — 高評価順、低評価順、新しい順、古い順
  • 新着作品 — 7日・14日・30日以内の新規配信作品
  • キーワード検索 — 日本語・英語タイトルで検索可能
  • 毎日自動更新 — JustWatch API と IMDb 公式データセットから毎日最新データを取得

サービスの目的

Netflixには膨大な作品が配信されていますが、評価の高い作品を効率的に見つけることが難しいという課題があります。本サービスは、世界最大の映画データベースであるIMDbの評価データを活用し、ユーザーが高品質な作品を素早く発見できるよう支援することを目的としています。

技術構成

  • フロントエンド：HTML/CSS/JavaScript（Vercel でホスティング）
  • バックエンド：Node.js + Express（VPS でホスティング）
  • データソース：JustWatch GraphQL API（作品情報）、IMDb公式データセット（評価スコア・ジャンル）
  • LINE連携：LIFF SDK を使用したLINEミニアプリ
